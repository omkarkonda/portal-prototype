Styleguide

A CSS style guide is a set of standards or guidelines that help developers maintain consistency, clarity, and maintainability in the way CSS code is written and structured across a project or organization. This is a live document, new styles/cmponents can be added as needed.

CSS Variables

CSS variables, allow developers to define reusable values for properties like colors, fonts, and sizes, making it easier to maintain and update styles across a project, enhance consistency and improve code readability.

css variables can be defined any where in the css file, and can be used in any selector. to maintain consistency, all css variables must be defined in the root element of the document. This is done for you and the root elemtn is available in base.css file.
